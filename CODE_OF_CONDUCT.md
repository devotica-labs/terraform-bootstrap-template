# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its Code of Conduct.

## In short

- Treat everyone with respect, regardless of background or experience level.
- Disagree on the technical merits — never on the person.
- Assume good faith. Ask before reaching a conclusion.
- Step away from a heated thread; come back when you can be productive.

## Scope

This Code of Conduct applies to all project spaces — issues, pull requests, discussions, code reviews, documentation, and any community channels operated under the `devotica-labs` GitHub organization. It also applies when an individual is officially representing the project in public spaces.

## Reporting

If you experience or witness behavior that is incompatible with this Code of Conduct, report it to the project maintainers at **conduct@devotica.com**. Reports are reviewed confidentially. You will not be retaliated against for reporting in good faith, even if the investigation does not conclude in your favor.

## Enforcement

Project maintainers are responsible for clarifying and enforcing standards of acceptable behavior. They may take any action they deem appropriate, including private warnings, public clarifications, temporary suspension from project spaces, or permanent removal from the community.

Decisions are made by the maintainer team. The full text of the Contributor Covenant — including the four-tier escalation ladder (Correction · Warning · Temporary Ban · Permanent Ban) — is the authoritative reference. See the link above.
